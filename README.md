## ✨ Features
* **Responsive Design:** Optimized for mobile, tablet, and desktop viewports.
* **About Me & Education:** A brief introduction and a timeline of my academic background.
* **Skills Section:** A visual breakdown of my technical proficiencies.
* **Projects Showcase:** Cards highlighting my latest work with descriptions.
* **Certifications:** A dedicated area displaying my earned credentials.
* **Contact Form / Links:** Clean layout for reaching out via email or social platforms.
